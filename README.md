## Technologies

Project is created with:
- react version: 18.2.0
- emotion/styled version: 11.11.0
-  axios version: 1.6.2
- react-router-dom: 6.20.1

## Setup

To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
